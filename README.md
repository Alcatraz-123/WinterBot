# 🔥 无尽冬日助手 — Android APK

自动挂机辅助，基于 AccessibilityService（无障碍服务），无需 Root。

## 快速获得 APK

### 方法一：GitHub Actions（推荐）

1. 把项目推送到 GitHub
2. 进入仓库 → Actions → Build APK → 自动开始编译
3. 编译完成后，点 **Artifacts** 下载 APK

### 方法二：Android Studio

1. 装 [Android Studio](https://developer.android.com/studio)
2. 打开本项目文件夹
3. Build → Build APK(s)
